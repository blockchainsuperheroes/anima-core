Agent-based vaults

ERC-4626 standardizes vault interfaces for deposits, withdrawals, and shares. This makes it a strong primitive for AI-managed or agent-mediated vault strategies.

ERC-8170 sits in front of that as the identity and continuity layer for the agent operating the vault. The agent's TBA holds vault shares, its certifications prove strategy competence, and its memory pointer tracks the portfolio thesis.

This creates a new primitive: verifiable agent fund managers. Users can inspect the agent's certification level, on-chain track record (via TBA), and lineage (is this a proven clone of a high-performing agent?) before delegating capital.
